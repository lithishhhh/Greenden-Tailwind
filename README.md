# 🌿 GreenLeaf - Plant Selling Website

GreenLeaf is a modern, responsive plant selling website built using HTML and Tailwind CSS. It features a clean and user-friendly interface that allows visitors to explore a wide variety of plants, learn about the brand, and easily get in touch.

## ✨ Features

- 🌱 Responsive design for all devices
- 🏡 Modern and attractive landing page
- 📦 Product showcase section
- 💚 About Us section
- ⭐ Customer reviews
- 📞 Contact section
- 🚀 Fast-loading and clean UI
- 🎨 Built with Tailwind CSS

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS
